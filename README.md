## 🛰️ SYSTEM OVERVIEW

This is a premium, high-performance portfolio engineered to showcase elite software engineering capabilities. Built with a "Hardware-First" design philosophy, it rejects generic web aesthetics in favor of a strictly engineered, aerospace-inspired HUD (Heads-Up Display) experience.

### 🛠️ CORE TECHNOLOGY STACK

- **Engine:** [Next.js 15+](https://nextjs.org) (Strict App Router Architecture)
- **Logic:** [TypeScript](https://www.typescriptlang.org/) (100% Type-Safe / Zero `any`)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/) (Fluid Typography / Precision Spacing)
- **Physics & Motion:** [Framer Motion](https://www.framer.com/motion/) (Spring-based mechanical animations)
- **Fluidity:** [Lenis](https://lenis.darkroom.engineering/) (High-precision smooth scrolling)
- **Environment:** [HTML5 Canvas](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) (Deep space parallax dynamics)

## 📡 KEY SYSTEMS & TELEMETRY

### 1. Dynamic Circuit Routing

A continuous, unbroken vertical trace line anchors the entire experience, dynamically routing between the center and the left side of the screen as you scroll. It represents the physical "interconnectedness" of the system.

### 2. Telemetry Navigation

A persistent top-level dashboard that tracks live scroll data:

- **ALTITUDE:** Mapped to current scroll position (0 - 400,000 FT).
- **VELOCITY (MACH):** Mapped to real-time scroll speed/velocity.
- **NETWORK GRAPH:** Live mouse-tracking SVG visualizer.

### 3. System Boot Sequence

An immersive initialization sequence that "decrypts" the UI, simulating a kernel boot process with real-time logs and hex-code generation.

### 4. Admin Terminal `[SYS.CMD]`

A fully functional, CRT-styled root terminal hidden behind the logo. It supports custom commands (`help`, `status`, `whoami`) and provides an interactive "God Mode" interface.

### 5. Hardware Visual Modules

Custom-coded SVG environments for key projects:

- **Qader Platform:** Live traffic log & topology visualizer.
- **Max CLI:** Interactive terminal kernel animation.
- **NanoManga:** Multi-modal AI pipeline simulation.

## 🚀 SYSTEM INITIALIZATION

### 1. Clone the Repository

```bash
git clone https://github.com/Abubakr-Alsheikh/abubakr-alsheikh-portfolio.git
cd abubakr-alsheikh-portfolio
```

### 2. Install Hardware Modules (Dependencies)

```bash
npm install
```

### 3. Start HUD Simulation (Development)

```bash
npm run dev
```

### 4. Compile Production Payload (Build)

```bash
npm run build
```
